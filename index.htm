O que é Tetris:

Tetris é um jogo eletrônico criado em 1985. O objetivo dele é empilhar blocos chamados tetrominós que descem no visor, em que você, o jogador, deve completar linhas horizontais com eles. Quando uma linha horizontal é completa ela é apagada e pontos serão acrescentados.

Como jogar o jogo:

1. Clique com botão esquerdo do mouse no visor acima para começar o jogo Tetris

2. Use "W","A","S" e "D" para girar e mover as peças que cairem

3. Use "↑" e "↓" para controlar a velocidade em que as peças caem

4. Pressione "Esq" para sair do jogo
